# my-codex-test

This repository includes a simple vertical shooting game for iPhone using SpriteKit. The game is located in the `iOSVerticalShooter` folder.

## Building the iOS Game
1. Open Xcode and create a new *Game* project using SpriteKit.
2. Replace the default `GameScene.swift` and `GameViewController.swift` with the files from the `iOSVerticalShooter` folder.
3. Add any assets you want for the player and enemy sprites.
4. Build and run on an iOS device or simulator in portrait orientation.
